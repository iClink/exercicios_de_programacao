Neste projeto foi feito um simples conversor de moedas, baseada na API de taxa de câmbio da "Exchange Rates API", onde foi feito com alguns recursos do GSON, como ensinados pela plataforma.
Ele adquire a moeda de origem, dado pelas opções, em seguida le o valor, e então para qual moeda deseja realizar a conversão.
Não houve problemas, o maior "desafio" em si foi ler a documentação para implementar sem problemas.
